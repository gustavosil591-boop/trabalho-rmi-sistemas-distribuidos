# trabalho-rmi-sistemas-distribuidos
Instruções de Execução 

Baixar a biblioteca GSON: wget (essa foi a biblioteca que usamos) https://repo1.maven.org/maven2/com/google/code/gson/gson/2.10.1/gson-2.10.1.jar 

Compilar o Código: javac -cp .:gson-2.10.1.jar *.java 

Executar o Servidor (Terminal 1):  java -cp .:gson-2.10.1.jar Servidor 

Executar o Cliente (Terminal 2): java -cp .:gson-2.10.1.jar ClienteApp
